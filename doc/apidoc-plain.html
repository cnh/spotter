Issues / index

GET /issues/:lat/:lng/in/:area
Shows issue in :area around the given :lat and :lng

Examples

GET /issues/18/72/in/1000.json
200
[
  {
    "id": 17,
    "issue_type": "Drainage",
    "latitude": 18.975,
    "longitude": 72.8258,
    "description": "Severe drainage problem",
    "image_url": "/system/issues/images/000/000/017/thumb/test.png?1395822314",
    "created_at": "2014-03-26T08:25:15.000Z",
    "url": "http://test.host/issues/17.json"
  }
]

Params

  • area : optional [ Must be String ]

    The area within which issues need to be shown

  • lat : optional [ Must be String ]

    Latitude of the user

  • lng : optional [ Must be String ]

    Longitude of the user

Issues / show

GET /issues/:id/:lat/:lng
show issue and its distance from user

Examples

GET /issues/18/72/in/1000.json
200
{
  "id": 17,
  "issue_type": "Drainage",
  "latitude": 18.975,
  "longitude": 72.8258,
  "description": "Severe drainage problem",
  "status": "1",
  "image_url": "/system/issues/images/000/000/017/medium/test.png?1395822314",
  "created_at": "2014-03-26T08:25:15.000Z",
  "distance": 4228.382904109771
}

Params

  • id : optional [ Must be String ]

    The id of the concerned issue

  • lat : optional [ Must be String ]

    Latitude of the user

  • lng : optional [ Must be String ]

    Longitude of the user

Issues / create

POST /issues
create a new issue

Examples

POST /issues/18/72/in/1000.json
{
  "issue": {
    "issue_type": "Drainage",
    "description": "Severe drainage problem",
    "latitude": 18.975,
    "longitude": 72.8258,
    "status": "1"
  }
}
201
null

Params

  • issue_type : optional [ Must be String ]
  • latitude : optional [ Must be String ]
  • longitude : optional [ Must be String ]
  • description : optional [ Must be String ]
  • status : optional [ Must be String ]
