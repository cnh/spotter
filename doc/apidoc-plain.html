Issues / index

GET /api/issues/:lat/:lng/:area
Shows issue in :area around the given :lat and :lng

Examples

GET /issues/18/72/1000.json
200
[
  {
    "id": 2,
    "issue_type": "Drainage",
    "latitude": 18.975,
    "longitude": 72.8258,
    "url": "http://test.host/issues/2.json"
  }
]

Params

  • area : optional [ Must be String ]

    The area within which issues need to be shown

  • lat : optional [ Must be String ]

    Latitude of the user

  • lng : optional [ Must be String ]

    Longitude of the user

Issues / show

GET /api/issues/:id/:lat/:lng
show issue and its distance from user

Examples

GET /issues/18/72/1000.json
200
{
  "id": 2,
  "issue_type": "Drainage",
  "latitude": 18.975,
  "longitude": 72.8258,
  "description": "Severe drainage problem",
  "status": "1",
  "created_at": "2014-02-01T14:11:30.000Z",

  "updated_at": "2014-02-01T14:11:30.000Z",

Params

  • id : optional [ Must be String ]

    The id of the concerned issue

  • lat : optional [ Must be String ]

    Latitude of the user

  • lng : optional [ Must be String ]

    Longitude of the user
