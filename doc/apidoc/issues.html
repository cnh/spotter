GET /api/issues/:lat/:lng/:area
Shows issue in :area around the given :lat and :lng

Supported Formats

json, xml

Examples

GET /issues/18/72/1000.json
200
[
  {
    "id": 2,
    "issue_type": "Drainage",
    "latitude": 18.975,
    "longitude": 72.8258,
    "url": "http://test.host/issues/2.json"
  }
]

Params

Param name Description
area
optional

The area within which issues need to be shown


Value: Must be String
lat
optional

Latitude of the user


Value: Must be String
lng
optional

Longitude of the user


Value: Must be String
