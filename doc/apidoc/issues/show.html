GET /api/issues/:id/:lat/:lng
show issue and its distance from user

Examples

GET /issues/18/72/1000.json
200

Supported Formats

json, xml

Params

Param name Description
id
optional

The id of the concerned issue


Value: Must be String
lat
optional

Latitude of the user


Value: Must be String
lng
optional

Longitude of the user


Value: Must be String
